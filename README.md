# Linizen

A bridge plugin between [AdvancedSlimePaper](https://github.com/InfernalSuite/AdvancedSlimePaper) and the [Denizen](https://github.com/DenizenScript/Denizen).

## Requirements

- Server running [AdvancedSlimePaper](https://github.com/InfernalSuite/AdvancedSlimePaper) (1.21.11+)
- [Denizen](https://github.com/DenizenScript/Denizen) plugin

## Build

Requires Java 16+ and Maven.

```bash
mvn clean package install
```
